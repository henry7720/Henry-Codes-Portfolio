Fedora bootc Migration Project - Immutable, Reproducible Container-Image Based OS Build Tooling | GitHub

APRIL 2026 - PRESENT

  • Infrastructure as Code (IaC): Developed a declarative system management framework for Fedora Atomic distributions using bootc, successfully transitioning the OS lifecycle from traditional rpm-ostree layering to a container-native workflow.
  • DevOps & Automation: Authored robust Bash automation scripts to streamline OS rebuilds, including logic for image tag rotation (latest vs. previous) to ensure reliable system rollbacks and automated cleanup of unpinned container layers.
  • Systems Engineering: Engineered customized OCI images (Containerfiles) that automate the installation of hardware drivers, multimedia codecs, and third-party repositories while managing complex package swaps like toolbox for distrobox.
  • Tooling & Integration: Created a system diagnostic utility leveraging skopeo, podman, and jq to perform remote-to-local digest comparisons, providing automated notifications for base image updates on external registries.
  • System Configuration & Hardening: Implemented low-level OS optimizations through the container layer, including masking insecure services, configuring fcitx input frameworks, and managing persistent networking services like Tailscale.
  • Container Security: Defined and maintained secure image builds using official Fedora Kinoite OCI images as a base, ensuring all custom layers are compatible with the bootc specification.
